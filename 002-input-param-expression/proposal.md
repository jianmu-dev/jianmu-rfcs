# 概述

* 输入参数支持表达式关联

# 问题描述

* dsl中节点定义输入参数只能关联单个变量（如，密钥、全局参数、事件参数、任务输出参数），不支持多个变量拼接的表达式

# 约束条件

* 需要重构参数关联模型

# 解决方案

* dsl示例：
```
qiniu_upload:
  type: jianmu_runner_qiniu:0.1.2
  param:
    qiniu_upload_prefix: ${xxx.xxx}/${xxx.xxx}/
```
